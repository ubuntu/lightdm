We welcome improvements and bug fixes to LightDM. All changes should be proposed as [pull requests](https://github.com/ubuntu/lightdm/pulls).

We also invite you to join the [LightDM Matrix Room](https://matrix.to/#/#lightdm:matrix.org).

Substantial changes must have associated regression tests. If you are having trouble making a suitable test please propose your change anyway and we'll try and help you write a test or write a test if necessary.
